# openlibx402-langgraph

LangGraph integration for OpenLibx402 protocol in Python.

## Features

- LangGraph workflow support
- Protocol operations via LangGraph

## Installation

```bash
pip install .
```

## Usage

```python
from openlibx402_langgraph import OpenLibX402Graph
```
